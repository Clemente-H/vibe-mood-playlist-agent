# Mood Vibe Curator

> A music agent that understands your emotions better than Spotify's DJ.

[![Cloud Run Hackathon 2025](https://img.shields.io/badge/Cloud%20Run-Hackathon%202025-4285F4)](https://run.devpost.com)

---

## The Idea

Spotify DJ is... meh. It always plays the same rotation, doesn't understand your current vibe, and feels robotic.

**Mood Vibe Curator** is an intelligent agent with a beautiful visual interface that lets you:
- Select colors that represent how you feel
- Describe your mood with words
- Get perfect music automatically added to your Spotify queue

Simple, beautiful, and actually gets you.

---

## Planned Features

- Intuitive visual interface for mood selection
- Interactive color palette
- Text input for describing emotions
- Direct Spotify queue integration
- AI agents that analyze and curate music
